# DOTFILES

![Terminal](./screenshots/terminal.png)

## Purpose

Config files used by many softwares on UNIX systems. Use at your own risk.

* editorconfig
* git
* vim
* zsh

## Requirements

* `npm -g install instant-markdown-d` for the Vim `instant-markdown-vim` plugin to work.
